French Telex

French input for Keyman using Vietnamese-Telex-like postfix rules.

Examples: e1le2ve -> élève, garc1on -> garçon, No3el -> Noël.

Escape a trigger literally with backslash, for example e\1 -> e1 and \" -> ".

Common shortcuts expand on the second space, for example cest then two spaces -> c'est , yat then two spaces -> y-a-t'il , and Ya then two spaces -> Il y a .

Double quotes become French guillemets: "bonjour" -> « bonjour ».
